# Introduction

Welcome to your knowledge base. This is a sample article — edit or delete it and replace it with your own content.

## Getting started

Add new articles by creating `.mdx` files inside `src/content/docs/<category>/`.

Each article needs these frontmatter fields:

```yaml
---
title: Your Article Title
description: A short description shown in search results.
category: your-category
order: 1
---
```

The `category` field groups articles in the sidebar. The `order` field controls sort order within a category.
